The **Search Bar** is an input element that lets users enter a brief text on a single line to execute a search on the current page.

---

## Usage

The **Search Bar** appears in pages where users can filter and/or search an instance or any other element. Example: tables in products pages, Marketplace homepage.

---

## Behavior

The **Search Bar** is composed by a bar, a magnifying glass tilted left icon, and a standard placeholder text.

When users click on the bar, the cursor appears. When users type a text, the placeholder disappears, and the search starts, exhibiting on screen the matches.

Users are able to enter any type of text, including letters, numbers, and symbols.

## Content

By default, the search on Azion is executed to find an instance or element by name.

A standard text was defined to be within this element: **Search**.

### Best practices

#### Capitalization

- Use sentence case.

| ✅ Do | ❌ Don't  |
|---|---|
| *Search*  | *Search an application here* |

#### Exceptions

In case of search engine working with another behavior, the recommendation is following this structure: Search by [behavior]

| ✅ Do | ❌ Don't  |
|---|---|
| *Search by date*  | *Searching your timestamp* |
